Variables Variables are used to store information to be referenced and manipulated in a computer program. Variable could be exemplified as a bucket because you can store water in the bucket, change the water and take the water somewhere else.
Loops In computer programming, a loop is a sequence of instructions that is continually repeated until a certain condition is reached. If I turn on a water tap, the water will keep coming until I turn it off.
Conditionals Referring to an action that takes place only if a specific condition is met. Most programming languages use the word if for conditional expressions. If I don't eat breakfast, I will be hungry all day.
Functions In computer science, function composition (not to be confused with object composition) is an act or mechanism to combine simple functions to build more complicated ones. A function could be an 'eat' function where everytime I eat, I take the food,chew and digest it. So, every time I repeat the same thing.
